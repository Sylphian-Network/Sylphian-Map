What is a map marker?

Map markers are locations on the community map. Each marker includes some required details and several optional extras to better describe the place or event.

Required fields

  • Title – The name of the place or event.
  • Type – Specify what kind of marker it is (e.g. event, business, member).
  • Latitude & Longitude – The coordinates where the marker will appear.

Optional fields

  • Content – A description, notes, or further details about the marker.
  • Address – Street address or location reference.
  • Icon / Variant / Colour – Customise how the marker appears on the map.
  • Marker colour – Highlight different categories with distinct colours.
  • Create a thread – Automatically start a discussion linked to this marker.
  • Lock thread replies – Prevent replies if you’d like the thread to be read-only.
  • Start / End date – Show when an event begins and finishes.

Example

Adding a marker for “Community Meet-up” might look like this:

  • Title: Community Meet-up
  • Type: Event
  • Latitude/Longitude: 51.5072, -0.1276
  • Content: Join us for coffee and a chat!
  • Start Date: 5 October 2025, 6:00pm
  • End Date: 5 October 2025, 7:00pm

Suggesting a marker

Suggesting a marker is simple: go to the community map and click the Suggest marker button. A form will appear where you can enter the details of the place or event you’d like to add.

After adding all the details, click Save to submit your marker for review. A moderator (or another approved user) will review your submission before it appears on the map.

Event markers

Event markers behave a little differently from other types of markers. They can animate on the map to make it easier to spot upcoming or ongoing events:

  • Active events – Events currently in progress will pulse on the map, drawing attention so you can find them at a glance.
  • Upcoming events – Events starting within the next 7 days will bounce gently to show they’re about to happen.
  • Past events – When an event has ended, its marker will disappear from the map (or be hidden automatically during clean-up).

This makes it easy to see not only where events are happening, but also which ones you can still attend soon.

Events widget

The events widget gives you a quick way to see what’s happening in the community without needing to browse the whole map. It lists the most recent event markers that are currently active or coming up soon.

  • Active events – If an event is in progress right now, it will appear at the top of the widget.
  • Upcoming events – Events that are starting soon will also be shown, ordered by start date.
  • Quick links – Each event entry includes a link to the full discussion thread if one was created.

This makes the widget a handy “at a glance” view of what’s happening in the community.

FAQ

Can I edit my marker after submitting?
Not directly, you can request an update from a moderator or suggest a new version.

Why isn’t my event showing on the map?
It may be waiting for approval or the event date may have passed.
